#  Универсальный калькулятор на Flutter

Многофункциональное приложение-калькулятор с красивым пастельным дизайном, включающее математический калькулятор, метрический и весовой конвертеры.

##  Возможности

###  Математический калькулятор
- Базовые арифметические операции (+, -, ×, ÷)
- Поддержка десятичных чисел
- Ввод с физической клавиатуры
- Очистка и удаление последнего символа
- Удобный интерфейс в пастельных тонах
- Компактные кнопки оптимизированного размера

###  Метрический конвертер
- Конвертация между различными единицами длины:
  - Миллиметры (мм)
  - Сантиметры (см)
  - Метры (м)
  - Километры (км)
  - Дюймы
  - Футы
  - Ярды
  - Мили
- Двусторонняя конвертация (ввод в любое поле)
- Мгновенное преобразование при вводе
- Кнопка быстрой смены единиц местами

### Весовой конвертер
- Конвертация между единицами веса:
  - Миллиграммы (мг)
  - Граммы (г)
  - Килограммы (кг)
  - Тонны (т)
  - Унции
  - Фунты
  - Стоуны
- Двусторонняя конвертация
- Автоматический пересчет

##  Дизайн

Приложение выполнено в приятных пастельных тонах:
-  Розовый (математический калькулятор)
-  Голубой (метрический конвертер)
-  Зеленый (весовой конвертер)
-  Желтый (кнопки операций)
-  Фиолетовый (цифры)

##  Технологии

- **Flutter** - UI framework
- **Dart** - язык программирования
- **Material Design 3** - дизайн-система

##  Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/calculator_app.git
cd calculator_app
```

### 2. Установка зависимостей

```bash
flutter pub get
```

### 3. Запуск приложения

```bash
# Запуск на подключенном устройстве
flutter run

# Запуск в веб-браузере с мобильным режимом
flutter run -d chrome

# Запуск на конкретном устройстве
flutter run -d <device_id>
```

### 4. Сборка APK

```bash
flutter build apk --release
```

### Как использовать
Математический калькулятор
Выберите вкладку "Математика"

Вводите числа кнопками или с клавиатуры

Используйте операции +, -, ×, ÷

Нажмите = для получения результата

Кнопка C очищает всё, ← удаляет последний символ

Конвертеры
Выберите нужный конвертер (Метрика или Вес)

Введите значение в верхнее поле

Выберите единицы измерения из выпадающих списков

Результат появится автоматически

Можно вводить значения в нижнее поле для обратной конвертации

Используйте кнопку ⇅ для быстрой смены единиц местами
